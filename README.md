# Check Receipt 

透過程式進行爬蟲抓取政府官方最新發票號碼，可供使用者進行輸入並比對最新一期獎號，查看是否中獎。 

Client : HTML、CSS、JavaScript、Pure.CSS 

Server : Node.js

# Use Step 

* Step 1 
```bash
$ git clone https://github.com/Samuelchi861008/Check-Receipt.git
``` 

* Step 2 
```bash
$ cd Check-Receipt
``` 

* Step 3 
```bash
$ npm install
``` 

* Step 4 
```bash
$ node app.js
``` 

* Step 5

Go to browser and type `localhost:3000`
